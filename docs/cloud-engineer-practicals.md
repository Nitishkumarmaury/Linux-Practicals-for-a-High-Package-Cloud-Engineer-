## 1. Linux & Networking Fundamentals
### Goal
Master Linux administration and networking concepts essential for cloud infrastructure.

### Basic Skills
- Shell scripting (bash/powershell)
- File system operations
- User management
- Network commands
- SSH configuration

### Hands-on Labs
1. Basic Level
   - Create and manage users
   - Configure SSH key authentication
   - Set up static IP in VM
   - Basic firewall configuration

2. Intermediate Level
   - Network traffic analysis with tcpdump
   - Advanced firewall rules
   - DNS troubleshooting
   - Routing configuration

3. Advanced Level
   - Automated OS hardening
   - Network simulation (packet loss/latency)
   - Security compliance scripts

### Project Deliverable
Create a GitHub repository containing:
- Automated VM provisioning scripts
- Security hardening playbook
- Network diagnostics toolkit
- Documentation with common troubleshooting scenarios
