# Cloud Engineering Labs

This directory contains hands-on labs for each practical in the cloud engineering career path. Each lab includes starter code, step-by-step instructions, and expected deliverables.

## Lab Structure
Each practical has its own directory with the following structure:
```
practical-XX/
├── README.md           # Detailed instructions and steps
├── code/              # Starter code and templates
├── scripts/           # Helper scripts and utilities
└── solutions/         # Example solutions (reference only)
```

## Getting Started
1. Clone this repository
2. Complete labs in order (01 → 10)
3. Each lab builds on skills from previous labs
4. Create a new branch for each lab
5. Follow the README in each lab directory

## Prerequisites
- Basic command line knowledge
- Git basics
- Text editor (VS Code recommended)
- Free tier accounts on AWS/Azure/GCP
